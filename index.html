Self-Balancing Toy: MATLAB Simulation Project

Team: SU-57

Members: Samridhi Shukla & Shreejal Tiwari

🔬 Project Overview

This project explores the physics and mechanics behind self-balancing toys using MATLAB simulations. It includes nonlinear dynamics, energy analysis, damping effects, and phase space visualization.

📄 Key Files

  • self_balancing_toy_nonlinear.m
  • self_balancing_toy_simulation.m
  • Presentation Script
  • Detailed Project Report (PDF)

🎯 Objective

The goal is to simulate and understand the dynamics of self-balancing mechanisms through modeling, equations of motion, and simulation plots using MATLAB.
